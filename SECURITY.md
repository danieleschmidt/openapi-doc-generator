# Security Policy

## Reporting Security Vulnerabilities

**Contact**: security@terragonlabs.com  
**Response Time**: Within 48 hours  
**Process**: Responsible disclosure

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0   | :x:                |
